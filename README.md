# AI-Powered Customer Support Ticketing System

## 📌 Project Overview

The AI-Powered Customer Support Ticketing System is a Salesforce CRM application developed to simplify and automate customer support operations. The system enables organizations to efficiently create, track, prioritize, and resolve customer issues using Salesforce's standard features along with automation tools.

This project integrates Salesforce Administration concepts such as Custom Objects, Validation Rules, Flows, Reports, Dashboards, Profiles, Permission Sets, and Users to build a centralized customer support management solution.

---

## 🎯 Objectives

- Automate customer support processes.
- Improve ticket tracking and management.
- Prioritize customer issues efficiently.
- Reduce manual work using Salesforce automation.
- Generate reports and dashboards for better decision-making.
